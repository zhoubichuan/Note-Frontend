1、弹性盒子

#one .father{ width: 200px; height: 200px; border: 1px solid black; display: flex; flex-direction: row; justify-content: center; align-items: center; }
#one .children{ height: 50px; width: 50px; background-color: red; }

2、translate

#two .father{ width: 200px; height: 200px; border: 1px solid black; position: relative; }
#two .children{ height: 50px; width: 50px; background-color: red; position: absolute; top: 50%; left: 50%; transform: translate(-50%,-50%); }

3、绝对定位

#three .father{ width: 200px; height: 200px; border: 1px solid black; position: relative; }
#three .children{ height: 50px; width: 50px; background-color: red; position: absolute; top: 0; left: 0; right: 0; bottom: 0; margin: auto; }

4、table-cell

#four .father{ width: 200px; height: 200px; border: 1px solid black; display: table-cell; text-align: center; vertical-align: middle; }
#four .children{ height: 50px; width: 50px; background-color: red; display: inline-block; }

5、行内块元素

#five .father{ width: 200px; height: 200px; border: 1px solid black; text-align: center; overflow: auto; }
#five .father:after{ content: ""; display: inline-block; vertical-align: middle; height: 100%; margin-left: 0.25em; }
#five .children{ height: 50px; width: 50px; background-color: red; max-width: 99px; display: inline-block; vertical-align: middle; }

6、负外边距(只能水平居中)

#five .father{ width: 200px; height: 200px; border: 1px solid black; text-align: center; overflow: auto; }
#five .father:after{ content: ""; display: inline-block; vertical-align: middle; height: 100%; margin-left: 0.25em; }
#five .children{ height: 50px; width: 50px; background-color: red; max-width: 99px; display: inline-block; vertical-align: middle; }
